# Personal Site 90s

Next.js 16 + TypeScript で作った、90年代風の個人サイトです。

## ページ構成

- `/` トップ
- `/profile` プロフィール
- `/links` リンク集
- `/works` 作品
- `/contact` 連絡先

## 開発

```bash
pnpm install
pnpm dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 検証

```bash
pnpm lint
pnpm build
```

## デザイン方針

- 強い枠線とシンプルな配色
- 小さめタイポグラフィ
- テーブルとリスト中心の情報設計
- ちょっと懐かしいテキスト演出
